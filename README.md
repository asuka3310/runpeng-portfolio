# Runpeng Ma — Portfolio

Analog IC & MEMS Engineer portfolio · 中英雙語

Live: https://runpeng.autumnsnow.work

## Stack

- Astro 4 + Tailwind CSS 3 + TypeScript
- i18n: zh-TW (default) / en
- Self-hosted Google Fonts via `@fontsource`

## Develop

```bash
npm install
npm run dev      # http://localhost:4321
npm run build    # outputs to dist/
npm run preview
```

## Deploy

Vercel auto-deploys on push to `main`.
